# Light Project

A light automation agent run every OS.

Which will support kafka / activemq as event / message channel.

# Arch
* LightAgent, automation client
* LightProxy, proxy channel
* LightServer, automation server

# Light Proxy Feature
* 支持 HTTP 请求代理
* 多个线程响应用户请求
* 支持 SOCKS5
